## Getting Started
This repo holds all the code for the Medium articles I write. To get started:

1. Clone the repo
```bash
git clone https://github.com/coolinmc6/medium.git
```

2. Navigate to the directory
```bash
cd medium
```

3. Install dependencies
```bash
npm install
```

4. Run the development server
```bash
npm run dev
```
